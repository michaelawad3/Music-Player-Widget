# MusicPlayerWidget

## Setup Instructions

1. Run `npm install` (installs Express)
2. Replace `YOUR_SPOTIFY_CLIENT_ID` in `spotify.js` with your real Client ID
3. Make sure your Spotify App's Redirect URI is set to: `http://127.0.0.1:8080`
4. Run the app with:
   ```
   npm run start
   ```

It will open a window and initiate Spotify login via OAuth.
